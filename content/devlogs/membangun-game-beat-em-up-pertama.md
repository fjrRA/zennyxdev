Game pertama Zennyx Interactive Studio saat ini masih menggunakan judul sementara: Untitled Andara Beat 'em Up.

Secara konsep, game ini adalah side-scrolling action beat 'em up berlatar Kota Andara, sebuah kota fiksi bernuansa urban yang terinspirasi dari kehidupan jalanan, konflik manusia, gang sempit, kota malam, dan tekanan sosial di lingkungan perkotaan.

Kami memilih beat 'em up karena genre ini cocok untuk tahap awal Zennyx. Gameplay-nya bisa dimulai dari fondasi kecil: karakter bergerak, menyerang, menghadapi musuh dasar, lalu menyelesaikan stage sederhana. Dari fondasi itu, sistem bisa dikembangkan secara bertahap tanpa harus langsung membuat dunia yang terlalu besar.

Fokus awal kami bukan membuat game yang langsung besar. Fokus awal kami adalah membuat prototype yang playable.

Untuk prototype pertama, target kami sederhana:

* Player bisa bergerak dengan nyaman.
* Player bisa melakukan basic attack.
* Ada simple combo.
* Ada musuh dasar.
* Ada health system.
* Ada stage kecil untuk testing.
* Ada kondisi stage clear.
* Ada UI dasar untuk combat.

Beberapa fitur sengaja kami tunda, seperti open world, multiplayer, RPG kompleks, skill tree besar, quest bercabang, dan cutscene panjang. Fitur-fitur itu mungkin menarik, tetapi belum cocok untuk tahap awal.

Kami ingin game pertama ini menjadi bukti bahwa Zennyx bisa membangun sesuatu yang nyata. Kecil dulu, tapi playable. Sederhana dulu, tapi jelas arahnya.

Devlog berikutnya akan digunakan untuk mencatat perkembangan prototype, keputusan desain, kendala teknis, dan perubahan scope selama proses development berjalan.
