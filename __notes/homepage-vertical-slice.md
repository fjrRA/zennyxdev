# Zennyx Homepage Vertical Slice

## Design north star

Zennyx should feel like an independent game zine and working studio from Indonesia.
The interface borrows from handmade posters, production sheets, pinned notes, and game credits—not from portfolio grids, dashboards, or SaaS landing pages.

### Must feel

- Game-first and media-first
- Small, honest, active studio
- Grounded, human, and warm
- Modular enough for future games to carry their own art direction
- Slightly imperfect, tactile, and assembled by people

### Must not feel

- Like the Fajar Works portfolio
- Cyberpunk, terminal-inspired, or HUD-like
- Like a generic AI/SaaS landing page
- Rounded card systems, pill controls, bento layouts, or fake product metrics
- Bigger or more established than the studio really is

## Homepage content map

1. Navigation
   - Zennyx identity
   - Games, Devlog, Studio, Contact
2. Hero / current production
   - Studio positioning
   - Current game title
   - Honest WIP media state
   - Current stage and core project metadata
3. Current game overview
   - Andara setting
   - Production status
   - Prototype focus
   - Next milestone
4. Development journal
   - Three latest notes
   - Direct path to the full archive
5. Studio approach
   - Short statement
   - Four working principles
6. Founders
   - Fajar and Faiz
   - Current roles and short bios
7. Follow the build
   - Devlog CTA
   - Contact path
8. Footer
   - Concise studio description
   - Navigation and active channels

## Media rule

Until real gameplay captures, blockouts, GIFs, or concept art are available, the homepage uses an explicitly labelled work-in-progress media frame. It must be replaced by real development material rather than generated artwork.
